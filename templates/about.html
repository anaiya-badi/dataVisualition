Welcome to My Website

This is a data visualization project aiming to show common targets of suicide bombers in different countries.

Suicide bombing is an attack carried out by someone who intends to kill other people along with themselves with a kind of bomb.

Data pipeline

After dowloading the csv from CORGIS I followed these steps:

  • In the sheets I combined each column with a high and low estimate into averaged columns.
  • I then deleted all columns other than the averaged, year, country, city, location, latitude, longitude, target description, target type, and group name columns.
  • I downloaded this edited version as a csv, and reformatted the data in python to a json, each attack being a nested dictionary with each data point about that attack being a key element pair.
  • Finally, I downloaded an alpha 2 code to country dictionary (linked here), deleted all columns but alpha 2 and countries in Google Sheets, and created a json linking each country to it's code.

    Map design process

    After figuring out latitude and longitude, points showed up in a gridlike pattern on the map, as shown above. The issue was that there was a lot of overlap of dots, so you couldn't tell how many there were supposed to be. This led me to scrapping that idea and focusing instead on displaying bomb targets using color, rather than number of attacks.

    Individual country infographic design process

    I decided that as I was focusing on three options, civilian, security, or political targets, a bar chart would make more sense than a line graph.
